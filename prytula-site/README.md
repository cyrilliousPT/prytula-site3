# Новый сайт Cyril Prytula

Это статический сайт: его можно бесплатно разместить на Cloudflare Pages и подключить домен `prytula.eu`.

## Как опубликовать бесплатно

1. Создайте пустой репозиторий GitHub с именем `prytula-site`.
2. Загрузите в него содержимое этой папки.
3. В Cloudflare откройте **Workers & Pages → Create application → Pages → Connect to Git** и выберите репозиторий.
4. Для Build command оставьте пустое поле, а для Build output directory укажите `/`.
5. После первой публикации откройте **Custom domains** и добавьте `prytula.eu` и `www.prytula.eu`. Cloudflare подскажет, какие DNS-записи изменить у регистратора домена.

## Как менять тексты

До настройки CMS проще всего открыть файл `data/site.json` прямо на GitHub: нажмите карандаш, измените текст, затем **Commit changes**. Cloudflare автоматически опубликует изменения через несколько секунд.

Папка `admin/` оставлена для более удобной редакторской панели Decap CMS. Для её подключения требуется один раз создать GitHub OAuth App и заменить строку `repo` в `admin/config.yml` на адрес вашего репозитория. Без этой настройки сайт и ручное редактирование `site.json` работают полностью.
